# 90_essays

90_essays の入口ページです。
この章は、教科書本編で学んだ論理学の道具を、
計算・ソフトウェア・データ処理・形式検証へつなぐための**橋渡しノート**です。

本編が「理論の基礎体力」を養う場所だとすると、
90_essays は「どこで役に立つか」を具体化する場所です。

## この章で扱うこと
- 本編概念を別分野へ接続する見取り図
- 学習モチベーションを作る軽量トピック
- 後続の深掘り（CS・PL・DB・検証）につながる導線

## こんな人におすすめ
- 論理学を学んでいるが、実務との接点が見えにくい人
- 計算機科学の理論を「用途ベース」で掴みたい人
- 本編の難所に疲れたとき、短い応用ノートで視点を切り替えたい人

## 読み方
- 本編（`00_orientation`〜`05_modal_and_nonclassical`）と並行して読んで構いません。
- 関心テーマを1本読み、必要に応じて本編へ戻る往復学習を推奨します。
- 各ノートは「導入 → 論理との接続 → 典型例 → つまずきポイント」の順で整備します。
