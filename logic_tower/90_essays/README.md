# 90_essays

90_essays の入口ページです。
本編で学んだ論理学の道具を、計算・ソフトウェア・データ処理の文脈へ横展開する短編ノート群です。

## この章で扱うこと
- 本編の概念を別分野へ接続する見取り図
- 学習モチベーションを作る軽量トピック
- 後続の深掘り学習につながる導線

## 読み方
- 本編（`00_orientation`〜`05_modal_and_nonclassical`）と並行して読んでよい章です。
- まず関心のあるテーマを1本読み、必要に応じて本編の該当章へ戻る運用を推奨します。
