# columns_and_qa

`columns_and_qa` の入口ページです。
このディレクトリでは、力学学習でつまずきやすい話題を Q&A と短いコラムで分割管理します。

## この章の目的

- 本編で学んだ式を、問題選択の判断基準へ落とし込む。
- 「なぜその保存則を使うのか」を言語化する。
- 計算ミスを減らすためのチェック観点を定着させる。

## この章で扱うこと

- 系の切り方
- 保存則の使い分け
- ガウス積分の位置づけ

## 読み進め方

1. まず `01_how_to_choose_system.md` で系の設定を確認する。
2. 次に `02_momentum_vs_energy.md` で保存則の優先順位を整理する。
3. 最後に `03_gaussian_integral_positioning.md` で補助テーマの位置づけを確認する。
