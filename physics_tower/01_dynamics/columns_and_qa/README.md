# columns_and_qa（力学コラムとQ&A）

このページは、力学で起こりやすい誤解を整理する補助ページです。

## 1. Q&A

### Q1. 外力ゼロと力ゼロは同じか

同じではありません。外力ゼロは

$$
\sum \mathbf{F}_{\text{ext}}=\mathbf{0}
$$

を意味し、内部力は存在してよいです。

### Q2. どの保存則を優先するべきか

衝突瞬間は運動量、位置と速度の変換はエネルギー、回転軸周りは角運動量が第一候補です。

### Q3. ガウス積分は力学で使うか

通常の初等力学では必須ではありません。主な登場場面は

- 統計力学の速度分布
- 拡散方程式
- 波束解析

などで、力学本編では補助的扱いです。

## 2. コラム：系の切り方

1つの物体だけを見ると外力項が増えます。複数物体をまとめると内部力が消え、保存則が使いやすくなります。

## 3. ミニ演習

1. 衝突問題で系の候補を2つ挙げ、どちらが楽か比較せよ。
2. 同じ問題を運動方程式と保存則で解き、式量を比較せよ。
3. ガウス積分が必要になる物理分野を1つ調べよ。
