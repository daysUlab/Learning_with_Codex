# columns_and_qa（力学コラムとQ&A）

このページは、`01_dynamics` 本編で学ぶ保存則・運動方程式について、

- つまずきやすい点
- 問題での判断基準
- 計算ミスを減らすチェック習慣

を補強するためのコラム集です。

## 1. 学習目標

1. 保存則の適用条件を言葉で説明できる。
2. 「どの式を使うか」を問題文から判断できる。
3. 典型的な誤りを先回りして回避できる。

## 2. Q&A（よくある疑問）

### Q1. 外力ゼロと力ゼロは同じか

同じではありません。外力ゼロは「系全体への外力の和がゼロ」を意味します。

$$
\sum \mathbf{F}_{\text{ext}}=\mathbf{0}
$$

内部力が存在していても、全運動量保存は成立します。

### Q2. 運動量保存とエネルギー保存は同時に使えるか

使える場合があります。孤立系で、かつ散逸が無視できるなら

$$
\mathbf{P}=\text{const.}
$$

と

$$
K+U=\text{const.}
$$

を同時に適用できます。

### Q3. どの保存則を先に使うべきか

- 衝突直前・直後は運動量保存
- 高さと速さの変換はエネルギー保存
- 回転軸まわりの変化は角運動量保存

を第一候補にすると整理しやすいです。

### Q4. ガウス積分は力学で使うか

初等力学では必須ではありません。主に

- 統計力学の分布関数
- 拡散・熱伝導の解析
- 波束やフーリエ解析

で重要度が高まります。力学章では「必要になったら接続する補助道具」という位置づけです。

## 3. コラム：系の切り方が半分を決める

同じ現象でも、系の取り方で解法難易度は大きく変わります。

- 単体で追うと外力項が多くなる。
- 合体系で追うと内部力が消え、保存則が前面に出る。

計算に入る前に「何を1つの系とするか」を決めるだけで、式が短くなり誤りも減ります。

## 4. コラム：検算テンプレを固定する

計算後は次の4点を固定チェックします。

1. 次元
2. 符号
3. 極限
4. 既知結果への還元

この4点を毎回回すと、途中で見落とした符号ミスや係数ミスを高確率で発見できます。

## 5. ミニ演習（5問）

1. 2体衝突問題で、単体系と合体系のどちらが適切かを理由付きで述べよ。
2. 「運動量保存は成立、エネルギー保存は不成立」となる例を1つ挙げよ。
3. 角運動量保存を使うと有利な状況を1つ挙げよ。
4. 計算後の次元チェックの具体例を1つ書け。
5. 力学章でガウス積分を先に学ばない理由を説明せよ。
